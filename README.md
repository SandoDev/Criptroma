# Criptroma

## About project

para hacer dibujos de los criptogramas creados
3 tipos de criptogramas:
- 6fragmentos
- 2fragmentos
- multidimensional

### Built With
* Language: Python3
* Database: MySQL
* libraries:
  * tkinter
  * pygame
